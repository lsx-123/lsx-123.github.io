博客搭建说明

安装构建环境

首先请安装Python，2和3都可以，之后初始化环境：

$ pip install mkdocs
$ python setup.py install

安装和使用Yeti主题，mkdocs.yml内修改theme: Yeti

$ pip install mkdocs-bootswatch

写博客

  • 创建新项目
$ mkdocs new [dir-name]
  • 项目布局
mkdocs.yml    # The configuration file.
docs/
​    index.md  # The documentation homepage.
​    ...       # Other markdown pages, images and other files.
  • 进入文件夹内写博客
$ gvim [dir-name]\docs\myblog.md
  • 调整配置
$ gvim mkdocs.yml
  • 查看帮助
mkdocs help

构建网页

初始化以后，每次构建只需要输入以下指令即可，构建后的文件在site文件夹内：

$ mkdocs build

如果只是想测试的话，请输入以下指令：

$ mkdocs serve

部署的网页可以通过127.0.0.1:8000或者localhost:8000来访问。
